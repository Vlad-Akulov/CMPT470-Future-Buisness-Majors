### Artifact Discovery and Verification

1. Find the Artifact


Artifact was found through a link in the research paper.

https://anonymous.4open.science/r/ECScan

Following the link on February 3, 2026 results in an expired repository (note the campus network sometimes blocks the site fully). 
When searching on the wayback machine, the only indexes were in February and March of 2025. These result in HTTP 302 error and an empty file error respectively. 
The wayback machine does show a redirect to: 

https://anonymous.4open.science/api/repo/ECScan/file/

Which then returns "This page is unavailable for archiving. The server returned code: because access requires authorization".

("ECScan" site:github.com) and ("Essense Clone Scanner" site:github.com) were searched, but there were no results that matched our given tool, nor any third party reuploads.

Given these results, we have concluded that this tool is not available for public use, and likely never was. 
As the publication date of the paper is February 2025, it seems there was never a working link, or one that died within weeks of publication. 
The paper notes that no other tools use their style of analysis, so using another tool may not be a useful endeavour.

Original recall and precision is mentioned in the paper:

At N=3 (their 'optimal' value):
Precision = .980
Recall (varies by project, Ant chosen as was best) = .821
