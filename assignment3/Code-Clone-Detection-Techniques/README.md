### Benchmarks Used

We tested ChatGPT on the BigCloneBench dataset using 20 randomly sampled Java function pairs. 

We opened a fresh chat in a private browser to ensure no prior memories affected results. A preliminary training prompt was provided, similar to the paper's technique.
This prompt was generated by ChatGPT, though edited down to reduce length.

```
You are a software engineering expert specializing in code clone detection.

Two Java functions are given below.

Your task is to determine whether they are code clones.

Clone definitions:

Type-1: Identical code except for whitespace or comments  
Type-2: Same structure but variable or identifier names changed  
Type-3: Mostly similar but with small additions, deletions, or modifications  
Type-4: Different implementation but same functionality  

If the functions implement the same behavior, they are clones.

Respond using ONLY one of the following:

Clone: YES
Type: Type1 / Type2 / Type3 / Type4

or

Clone: NO

Example 1

Function A:
int sum(int a,int b){ return a+b; }

Function B:
int sum(int x,int y){ return x+y; }

Answer:
Clone: YES
Type: Type2


Example 2

Function A:
for(int i=0;i<n;i++){ total+=a[i]; }

Function B:
int i=0;
while(i<n){
 total+=a[i];
 i++;
}

Answer:
Clone: YES
Type: Type4

Classify these snippets:
```

Each pair was manually prompted using a simplified instruction to answer YES or NO for functional similarity.

```
Are these two Java functions similar?

Respond ONLY with:
YES
or
NO

Function A:
{code1}

{code2}

Answer:
```

### Results Assessment

ChatGPT classified 19 of 20 pairs correctly, with only one false negative.
15 NOs were provided, 5 YES'. ChatGPT missed a single YES, and identified all NOs.

Accuracy : 0.95
Precision: 1.0
Recall   : 0.8
F1 Score : 0.89
